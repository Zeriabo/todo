# todo
Spring boot TODO application 

# Description 
Application that requires users to be logged in before they can call the APIs. One user can create multiple todo items and one todo item can only belong to a single user

# language 
Java 11 or above

# Framework 
Spring Boot

# Database 
PostgreSQL
